# Goals

Active goals across all areas of life. Not a wishlist -- things you're actually working on.

One file per goal, named like `<short-name>.md`. Move completed/abandoned goals to `goals/archive/`.

## What goes in each file
- What the goal is, in plain language
- Why it matters to you (the actual reason, not the polished version)
- How you'll know you've reached it (concrete success criteria)
- Deadline or timeline
- Current status
- Progress log (dated entries showing what you've done)
- Blockers
- Next concrete action

## Pushback rule
If you've added a goal and there's been no progress for 3+ weeks, I should call it out. Either restart it or kill it -- don't let zombie goals sit here making you feel guilty.
