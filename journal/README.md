# Journal

Free-form personal entries. Different from `daily_logs/` which is structured "what happened today" -- journal is "what's on my mind."

## What goes here
- Reflections
- Things you're processing
- Stuff you wouldn't put in a work log
- Wins, losses, frustrations, gratitude
- Half-formed thoughts you want to come back to

## Format
One file per entry, named `YYYY-MM-DD.md` or `YYYY-MM-DD-topic.md`. Or one big rolling file if you prefer. Whatever you'll actually do.

## Privacy
Some journal entries might belong in `private/` instead. The default here is "syncs to GitHub" -- if an entry is too personal for that, move it to `private/`.

## Pushback rule
If something's bothering you and you keep avoiding it in your daily logs, I should notice and ask about it directly. Avoidance is information.
