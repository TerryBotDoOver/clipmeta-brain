## ClipMeta Flight Deck UI redesign — shipped

Multi-session design sprint. Complete visual redesign of all 10 ClipMeta marketing surfaces with a cinematic "Flight Deck" aesthetic inspired by sci-fi HUD interfaces + drone footage aesthetic.

### What shipped (commit `80f75df`)

**Home** (`/`): Full rebuild. New sections added:
- Hero with HUD status chip ("SYSTEM · ONLINE | METADATA PIPELINE | READY"), gradient headline, `HeroVisual` client component (animated drone frame with orbiting keyword tokens, timecode, REC indicator)
- `StatsBar` pulling live Supabase counts (contributors, clips processed)
- `MetadataPreview` — the showpiece: drone clip placeholder + typewriter-animated title/description/50 keywords/category/location output panel with platform tabs
- `BeforeAfter` — manual workflow (8min/clip) vs ClipMeta workflow (30sec/clip), 16× faster
- Three-stage pipeline (INGEST/ANALYZE/DELIVER) with connector line
- ArcadeDemo wrapped in animated conic-gradient border
- Bento feature grid with 3D tilt on hover
- Comparison table (ClipMeta vs TagMyClip vs FreeMetadata vs Manual)
- Interactive `SavingsCalculator` — clip count slider → time saved / value / auto-picked plan / ROI multiple
- Final CTA with full aurora background

**Pricing** (`/pricing`): 4 plan cards restyled as glass-cards, Pro with violet glow ring + "MOST POPULAR" pill, compare table restyled, credit packs restyled, FAQ rebuilt with Q01/Q02 monospace numbering.

**Auth** (`/auth`): Two-column glass-card layout. Left: gradient headline + aurora glow corners. Right: form panel with backdrop blur.

**Blackbox** (`/blackbox`): Full redress with glass cards and gradient headline.

**Blog** (`/blog` + `/blog/[slug]`): Listing redesigned with "BLOG · FIELD NOTES" hero + glass post cards. Article reader got Flight Deck prose styling (violet links, aurora-friendly blockquotes, glass code blocks).

**Landing pages** (5): `/lp/stock-footage-metadata`, `/lp/bulk-keywording`, `/lp/blackbox-upload`, `/lp/reddit`, `/lp/social` — all wrapped in FlightDeckShell, HUD chips, glass cards, gradient CTAs.

**In-app pages** (`/(app)/*`): intentionally not touched. Dashboard / projects / review / settings stay utility-focused per design discipline (apps should get out of the way, not decorate).

### Architecture

**New:** `FlightDeckShell` (`src/components/landing/FlightDeckShell.tsx`) — reusable marketing-page wrapper providing:
- Fixed-position aurora mesh (page-global, no more "solid black bar" behind nav)
- Blueprint grid overlay
- Soft radial gradient near top (nav framing)
- Film grain fixed overlay
- Cursor-following `MouseGlow` (RAF-based, respects `prefers-reduced-motion`)
- Floating pill nav (rounded, translucent, backdrop-blur)
- Footer with "STATUS · OPERATIONAL" chip

**8 new components** in `src/components/landing/`:
- FlightDeckShell, HeroVisual, MetadataPreview, BeforeAfter, SavingsCalculator, StatsBar, MouseGlow, RevealOnScroll

**CSS additions** (~220 lines in `globals.css`): aurora keyframes, grid overlay, film grain, HUD chips, chromatic aberration, kinetic letter reveal, conic gradient borders, glass cards with 3D tilt, scan line sweep, scroll reveal, CTA shimmer, glow ring.

**Theme change:** `ThemeProvider` forced to `defaultTheme="dark" forcedTheme="dark"` (was `system`). Flight Deck is the ClipMeta look; no more light-mode flicker on visitors with light OS.

**next.config.ts:** Added `turbopack.root = process.cwd()` to pin module resolution (there was a stray `C:\Users\levib\package.json` 67-byte stub from March that was confusing Turbopack; moved to `_claude_moved_files/`).

### Design reference used
The redesign was informed by the CSV data library we extracted from UI UX Pro Max (github.com/nextlevelbuilder/ui-ux-pro-max-skill) and saved to `ClipMeta_Brain/design_reference/data/` earlier today — 30 CSVs covering 67 styles, 95 color palettes, 56 font pairings, 29 landing patterns, 161 UI reasoning rules, and 16 framework-specific guidance files. Picked from `landing.csv` (Hero + Features + CTA, Comparison Table + CTA, Before-After Transformation) and `styles.csv` (Glassmorphism, Aurora UI, HUD / Sci-Fi FUI, Chromatic Aberration).
