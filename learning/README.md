# Learning

Books, articles, courses, ideas you're chewing on, things you want to remember from what you've read or watched.

## Suggested structure
- `books/<book-title>.md` -- one file per book with notes, key takeaways, quotes
- `articles.md` -- shorter notes on articles/videos worth remembering
- `ideas.md` -- raw ideas you're playing with, even if half-formed
- `courses/` -- if you're taking online courses

## What I can do with this
- Resurface notes when relevant ("you read this thing about cron monitoring, here's the relevant bit")
- Connect ideas across sources
- Track what you've learned over time vs what's still on the to-read pile
