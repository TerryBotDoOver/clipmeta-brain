## What goes in each file
- Identity (name, email, country, language)
- Plan + subscription state (trial / active / canceled)
- Stripe customer ID + subscription ID
- Usage pattern (daily upload counts, deletion behavior, regen rate)
- Project history
- Conversion timeline / payment method on file
- What they're using ClipMeta for (workflow guess)
- Conversion likelihood / churn risk
- Any direct interactions (support tickets, emails, feedback)
- Things to remember when reaching out

## Why this exists
So when Levi mentions a customer by name, I have full context immediately. Also so the graph view shows real customer relationships, not just abstract "user count" numbers.
