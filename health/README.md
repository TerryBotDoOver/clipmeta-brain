# Health

Sleep, workouts, mood, energy, anything physical or mental you want tracked over time.

## Suggested files
- `baseline.md` -- your normal: typical sleep hours, resting energy, weight, anything you want a baseline on
- `sleep.md` or `sleep-log.md` -- daily sleep tracking if you want it
- `workouts.md` -- workout log (or one file per workout type)
- `mood.md` -- mood tracking if you want it
- `concerns.md` -- ongoing health concerns or symptoms you're tracking
- `meds.md` -- if applicable

## Privacy note
Specific medical details might be better in `private/` (gitignored) instead of here. You decide. The default for this folder is "fine to sync to GitHub" -- if it's not, move it.
