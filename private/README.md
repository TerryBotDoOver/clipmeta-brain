# Private

This folder is gitignored. Everything in here stays on whichever machine wrote it. It does NOT sync to GitHub or to the other laptop.

## What goes here
- Anything you wouldn't want a stranger to read
- Sensitive health details
- Specific financial data (account balances, tax IDs)
- Conflict with people (so it's not in a synced repo)
- Legal stuff
- Anything you want to write down but not store in the cloud

## Trade-off
The trade-off for privacy is no sync. If you write something in `private/` on the Dell, the Predator won't see it. If the Dell dies, that file is gone unless you have a separate backup.

## I can still read this
Within a session, I can read anything in this folder. The privacy is about it not leaving the machine -- not about hiding it from me.
